Он начинается с тега и заканчивается тегом . - содержит метаинформацию о документе, такую как название страницы, ссылки на внешние ресурсы (стили, скрипты и т.д.) и другие элементы, которые не отображаются на самой странице. - задает название страницы, которое отображается на вкладке браузера и используется поисковыми системами. <body> - содержит все видимые элементы веб-страницы, такие как текст, изображения, ссылки, формы и другие. Контент страницы всегда располагается внутри этого тега. Заголовки (<h1> - <h6>): Определяют заголовки различных уровней. <h1> используется для основного заголовка страницы, а <h6> для самого маленького заголовка. Абзац (<p>): Определяет абзац текста. Ссылка (<a>): Создает гиперссылку. Изображение (<img>): Вставляет изображение на страницу. <a href="https://example.com">Ссылка на сайт</a> <img src="image.jpg" alt="Описание изображения"> Атрибут href для тега <a> указывает URL-адрес ссылки. Атрибут src для тега <img> указывает путь к файлу изображения. Атрибут alt для тега <img> содержит текстовое описание изображения, которое отображается, если изображение не может быть загружено. Список <ol> <li>Интерны</li> <li>Ходячие мертвецы</li> <li>Кухня</li> <li>Сверхъестественное</li> <li>Наруто</li> </ol> <strong></strong> слово между будет жирным <p>Это <strong>слово</strong> да</p> <h1>Заголовок первого уровня</h1> <h2>Заголовок второго уровня</h2> <p>Это пример абзаца текста на веб-странице.</p> <strong>Жирный текст</strong> <em>Курсивный текст</em> <u>Подчеркнутый текст</u> <small>Мелкий текст</small> <pre>Текст с сохранением форматирования</pre> <blockquote>Цитата</blockquote> Основные теги для изображений и мультимедиа: Изображения (<img>): отображает изображение на веб-странице. Атрибут src указывает путь к файлу изображения, а атрибут alt добавляет текстовое описание, которое будет отображаться, если изображение не загрузится. Также можно использовать атрибуты width и height для определения размеров изображения. Видео (<video>): позволяет встраивать видео на веб-страницу. Атрибуты width и height указывают размер видеоплеера, а атрибут controls добавляет элементы управления. Тег <source> внутри тега <video> указывает путь к видеофайлу и его тип с использованием атрибутов src и type. Можно добавить несколько тегов <source> с разными типами видео, чтобы обеспечить совместимость с разными браузерами. Аудио (<audio>): позволяет встраивать аудиофайлы на веб-страницу. Атрибут controls добавляет элементы управления аудиоплеером. Тег <source> внутри тега <audio> указывает путь к аудиофайлу и его тип с использованием атрибутов src и type. Аналогично тегу <video>, можно добавить несколько тегов <source> с разными типами аудио для обеспечения совместимости с разными браузерами. Таблицы: <table>: создает таблицу. Атрибут border устанавливает толщину границы таблицы. <thead>: определяет группу заголовочных ячеек в таблице. <tbody>: определяет группу основных ячеек в таблице. <tr>: создает строку таблицы. <th>: создает заголовочную ячейку в таблице. <td>: создает стандартную ячейку таблицы. 2.
